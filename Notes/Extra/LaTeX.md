# LaTeX

\sum{} --> summation symbol --> $\sum{d}$

\int{} --> integral symbol --> $\int{dd}$

\iint{} --> double integral symbol --> $\iint{dd}$

\neq = --> not equal symbol --> $\neq $

\dfrac{}{} --> used to make a fraction --> $\dfrac{x}{y}$

\boxed{} --> will box the content --> $\boxed{\frac{c}{d}}$

\bar{} --> will put solid line over content --> $\bar{y}$

\hat{} --> will put ^ over content --> $\hat{y}$

\theta --> theta symbol --> $\theta$

\lambda --> lambda symbol --> $\lambda$

\pi --> pi symbol --> $\pi$

\alpha --> alpha symbol --> $\alpha$

\beta --> beta symbol --> $\beta$

\gamma --> gamma symbol --> $\gamma$

\Theta --> theta symbol for discrete math --> $\Theta$

\Delta --> triangle symbol --> $\Delta$

\reals --> real numbers symbol --> $\reals$

\N --> natural numbers symbols --> $\N$

\cnums --> complex numbers --> $\cnums$

\sqrt{} --> square root symbol --> $\sqrt{dd}$

\sqrt[]{} --> square root symbol with number on left part --> $\sqrt[n]{abc}$

prime symbol thing by just putting the ' next to the thing --> $f'$

\varOmega --> Omega symbol --> $\varOmega$

\bigcup --> union of sets --> $\bigcup$

\bigcap --> intersection of set --> $\bigcap$

\cap --> smaller version of \bigcap --> $\cap$

\cup --> small version of \bigcup --> $\cup$

\lfloor and \rfloor --> floor function symbol. Important to not need both of them --> $\lfloor{r}\rfloor$

\lceil and \rceil --> ceiling function symbol. Important to not need both of them --> $\lceil{x+5}\rceil$

put the \_ and a thing after it would be a subscript --> $g_x^3$

\ln --> ln symbol --> $\ln$

\sin --> sin symbol --> $\sin$

\cos --> cosin symbol --> $\cos$

\tan --> tan symbol --> $\tan$

\log --> log symbol --> $\log$

\lim --> limit word --> $\lim$

\infin --> infinite symbol --> $\infin$

\in --> element of --> $\in$

\notin --> not element of --> $\notin$

\exist --> there exist at least --> $\exist$

\forall --> all elements of --> $\forall$

\therefore --> therefore symbol --> $\therefore$

\varnothing --> empty set --> $\varnothing$

\overbrace{}^{} --> nice way to put a description on top of a section of code --> $\overbrace{a + v}^{\text{hello}}$

\underbrace{}`_`{} --> nice way to put a description under a section of code --> $\underbrace{a + v}_{\text{hello}}$

\to --> right pointing arrow --> $\to$

\gets --> left pointing arrow --> $\gets$

\leftrightarrow --> left and right arrow --> $\leftrightarrow$

\subset --> subset symbol --> $\subset$

\supset --> super set symbol --> $\supset$

\neg --> discrete math not symbol --> $\neg$

\ge --> greater than or equal to --> $\ge$

\le --> less than or equal to --> $\le$

\land --> discrete math and symbol --> $\land$

\lor --> discrete math or symbol --> $\lor$

\And --> write actual & symbol --> $\And$

\pm --> plus and minus symbol --> $\pm$

\approx --> approximate symbol --> $\approx$

\equiv --> equivalent symbol --> $\equiv$

`\\` --> this is like the newline symbol --> $\text{yes sit}\\\text{Right here}$

\text{} --> way to write actual text --> $\text{yes sir here}$

& --> left and right adjust content but can only be used on certain things and places.

Matrix --> need to have the content written between a `\begin{bmatrix}` and `\end{bmartix}` box. Then put al code between it. Use the & to separate the content. $ A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} $

\mod --> used to have the mod word --> $c\mod{e}$

\div --> for division symbol --> $\div$

\binom --> for making binomial --> $\dbinom{x}{d}$

\measuredangle --> make angle symbol with little arch inside --> $\measuredangle ABC = 90^\circ$

\angle --> make angle symbol --> $\angle A = 60^\circ$

\textcolor{}{} --> way to give color to sections of writing. First section must be the color coding --> $\textcolor{#ff0382}{x+y\div{\frac{r}{c}}}\textcolor{#04ff00}{+r}$

> [!NOTE]
>
> add `arc` in front of the sin, tan, and cos symbol to make it the arc version

> [!NOTE]
>
> Everything can have an {} added after it and it is a way to group things whatever is being applied like $g_{xxxx}$

> [!IMPORTANT]
>
> To actual use { or } symbols have to cancel them out by doing `\{` and `\}`
